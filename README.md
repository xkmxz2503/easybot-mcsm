# 一个适用于 EasyBot 的 MCSM 连接插件

本插件为 EasyBot 提供了与 MCSM 的连接功能。支持通过命令远程开启或关闭服务器，通过调用 MCSM 的 API 接口实现。

## 使用方法

1. 将插件文件放入 EasyBot 的插件目录。  
2. 在 EasyBot 中配置 MCSM 的服务器 URL、API 密钥、守护进程 ID 以及实例 UUID。  
3. 使用以下命令控制服务器：  
   - `开启 <服务器名称>`  
   - `关闭 <服务器名称>` 
   - `重启 <服务器名称>`  
   - `强制关闭 <服务器名称>`  


## 注意事项

- 请确保 MCSM 的 API 接口配置正确且可正常访问。  
- 插件需要网络权限才能调用 MCSM 的 API 接口。  
- 本插件仅提供远程开关服务器功能，不涉及其他管理操作。
